# DSA_exercises
This repository contains my personal solutions to exercises in **Data Structures and Algorithms**, developed during my studies at the University of Modena and Reggio Emilia.
The goal of this project is to strengthen my problem-solving skills and build a solid understanding of fundamental algorithmic techniques.

---

## 📚 Topics Covered

Currently, the repository includes:

* **Recursion**
* **Backtracking**

More topics  will be added progressively.
